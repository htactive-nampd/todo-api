# Run 
```
# Install dependencies
$ yarn install

# Serve on localhost:2500
$ yarn start

# build js file
$ yarn build
```
### End point:
```
get list:
GET:/ localhost:2500/api

get item by listId
GET:/ localhost:2500/api/ {listId}

add item
POST:/ localhost:2500/api

run test:
$ yarn get
$ yarn getList
$ yarn post
```
